Site Name

"Oni Samson Promise Portfolio"

This name reflects my personal brand as a web developer and designer. It is unique, memorable, and suitable for showcasing my professional work and attracting clients or employers.

Site Purpose

The purpose of this website is to present my portfolio of web development projects, share technical blog posts, and provide contact information for collaboration or employment opportunities. It also includes a contact form that saves input using JavaScript and localStorage for demonstration purposes.

Scenarios

  • Can I view a list of web development projects Oni Samson Promise has completed?
  • How can I contact Oni Samson Promise for freelance work or collaboration?

Color Schema

  • Primary Color: Dodger Blue (#1e90ff) - used for headers, buttons, and navigation background.
  • Text Color1: Dark Gray (#333333) - used for body text.
  • Background Color: White (#ffffff) - used for the default background of the site.
  • Dark Mode Background: Charcoal (#121212) - used when dark theme is toggled.
  • Text Color2: Deep White (#121212) - used for dark text.

Typography

  • Primary Font: Roboto, sans-serif
  • Usage: Headings, paragraphs, buttons, navigation
